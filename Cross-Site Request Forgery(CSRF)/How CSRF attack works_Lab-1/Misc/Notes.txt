##CSRF LAB 1
Objective: Demostrate how CSRFV attack works

Goal: Change victims email via CSRF attack

##Conditions for CSRF
1. Relevant action: An actionb that an attacker has a reason to want to induce.Usually state-changing actions e.g password changing
2. Cookie-based session handling: The application should solely rely on cookies to track which use has issued a request and no any other mechanism
3. no unpredictable request parameter: e.g password to change a password
4. The user must authenticated or logged in to the application- credentials are given
